## Bilibili客户端的ReactNative版本
---
目前进度：合并分支，现在使用到了Redux框架（全靠Vai先森），利于管理整体主题；将上方的导航选项卡换为使用react-native-scrollable-tab-view插件，更加友好流畅；

### 开发环境

+ PLATFORM:    Windows
+ RN:   React-Native 0.29
+ FRAME:   Redux
+ IDE:   Webstorm
+ VM:   Genymotion
